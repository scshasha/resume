## Installing

### Prerequisites

* [Node/NPM](https://nodejs.org/en/)
* [Angular/Angular CLI](https://cli.angular.io/)


Execute:
```sh
git clone https://github.com/scshasha/resume.git resume
cd resume
npm install
npm start
```

Now browse [http://localhost:4200](http://localhost:4200) :shipit:
